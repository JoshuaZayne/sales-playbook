# Permissions

## Allowed
- Read all files and directories
- Create new files and directories
- Edit and modify existing files
- Rename files and directories
- Move files and directories
- Copy files and directories
- Push and pull from remote
- Create and merge branches
- Create and close issues and pull requests
- Add dependencies and packages
- Run scripts and builds
- Commit changes

## Requires Approval
- **Deleting files or directories** — always ask before removing anything
- **Deleting branches** — always ask before deleting any branch
- **Force pushing** — always ask before force pushing
- **Dropping or removing data** — always ask before any destructive data operation
